# Автоматический деплой в GitHub

## Настройка завершена! ✅

Токен GitHub сохранен в файле `.github_token` (не коммитится в git).

## Использование

### 1. Отправить изменения в GitHub:
```bash
./push-to-github.sh
```

Скрипт автоматически:
- Проверит наличие изменений
- Добавит все изменения
- Создаст коммит
- Отправит в GitHub (без запроса пароля)

### 2. Полный деплой (GitHub + сервер):
```bash
./deploy.sh
```

Скрипт автоматически:
- Отправит изменения в GitHub
- Обновит код на сервере
- Пересоберет Docker образ
- Применит миграции БД
- Перезапустит веб-сервер

## Безопасность

- Токен хранится в `.github_token` (в `.gitignore`)
- Файл имеет права доступа 600 (только владелец)
- Токен не попадет в git репозиторий

## Если нужно обновить токен

1. Отредактируй файл `.github_token`
2. Вставь новый токен
3. Сохрани файл

## Проверка

Проверить, что все работает:
```bash
./push-to-github.sh
```

Если нет изменений, скрипт сообщит об этом.
