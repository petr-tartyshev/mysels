# Как посмотреть изменения перед деплоем

## Быстрые команды для просмотра изменений

### 1. Посмотреть последние коммиты
```bash
git log --oneline -10
```
Показывает последние 10 коммитов с кратким описанием.

### 2. Посмотреть изменения в последнем коммите
```bash
git show HEAD
```
Показывает полный diff последнего коммита.

### 3. Посмотреть изменения в конкретном файле
```bash
git show HEAD -- app/profile/page.tsx
```
Показывает изменения в конкретном файле в последнем коммите.

### 4. Посмотреть все измененные файлы
```bash
git diff HEAD~5 HEAD --stat
```
Показывает список файлов, измененных в последних 5 коммитах.

### 5. Посмотреть изменения в нескольких последних коммитах
```bash
git log -p -3
```
Показывает полный diff последних 3 коммитов.

### 6. Посмотреть изменения в конкретном коммите
```bash
git show <commit-hash>
```
Например: `git show 24e55c7`

### 7. Посмотреть, какие файлы изменены локально (еще не закоммичены)
```bash
git status
git diff
```

### 8. Посмотреть изменения между локальной и удаленной версией
```bash
git log origin/main..HEAD --oneline
```
Показывает коммиты, которые есть локально, но еще не отправлены на сервер.

### 9. Посмотреть изменения в конкретном файле между коммитами
```bash
git diff HEAD~3 HEAD -- app/profile/page.tsx
```
Показывает изменения в файле между 3 коммитами назад и текущим.

### 10. Красивый просмотр изменений
```bash
git log --graph --oneline --all -10
```
Показывает дерево коммитов с графиком.

## Полезные алиасы (можно добавить в ~/.gitconfig)

```bash
git config --global alias.lg "log --oneline --graph --all -10"
git config --global alias.last "log -1 HEAD"
git config --global alias.changes "diff HEAD~1 HEAD"
```

После этого можно использовать:
- `git lg` - красивый лог
- `git last` - последний коммит
- `git changes` - изменения в последнем коммите

## Просмотр изменений в IDE

Если используешь VS Code или другой редактор:
- Открой файл и посмотри изменения через Git панель
- Используй встроенный Git diff viewer

## Онлайн просмотр на GitHub

- https://github.com/petr-tartyshev/mysels/commits/main
- Нажми на коммит, чтобы увидеть изменения
